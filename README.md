## Project setup

```bash
$ npm install
```

## Install Nest CLI

```bash
$ npm i -g @nestjs/cli
```

## setup DB

```bash
$ docker-compose up -d
```


## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## If you want to fill your DB with pokemons, you can hit the seed endpoint in postman
```
http://localhost:3000/api/seed
```
